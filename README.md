
# Ulkomaalarit.fi

**Ammattitaitoista ulkomaalausta Oulun seudulla**

Ulkomaalarit on oululainen yritys, joka tarjoaa laadukasta ulkomaalausta omakotitaloihin, taloyhtiöihin ja liikekiinteistöihin Pohjois-Pohjanmaalla. Palvelumme perustuvat ammattitaitoon, selkeyteen ja avaimet käteen -periaatteeseen.

## Palvelut
- Omakotitalojen ulkomaalaus
- Taloyhtiöiden ja liikekiinteistöjen maalaus
- Julkisivujen puuverhouksen uusiminen
- Timpurintyöt (laudoitukset, korjaukset)
- Metalli-, betoni- ja kivipintojen maalaus

## Tietoa projektista
Tämä projekti sisältää Ulkomaalarit.fi-sivuston lähdekoodin. Sivusto on rakennettu HTML- ja CSS-teknologioilla ja julkaistu GitHub Pagesin kautta. Sivustolla on käytössä hakukoneystävälliset meta-tiedot, schema.org-rakenteinen data ja EmailJS-yhteydenottolomake.

2. Commitoi ja pushaa GitHubiin
